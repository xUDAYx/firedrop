Age and Gender Prediction with Deep Learning!

This awesome app uses deep learning magic to predict someone's age and gender based on a x-ray image! Just upload a photo, and our clever models will do their best detective work to unveil the mystery.

The lung scans used in this project come from a Publicly available Datset Link for Dataset. It contains approximately 10,700 scans for training and 11,700 scans for testing. This dataset was part of a competition held by The Radiology and Diagnostic Imaging Society of São Paulo (SPR) with Amazon Web Services.

installation instructions

Downloading and Preparing the Folder

Download the required file from the provided source. Launch Visual Studio Code Navigate to the location where you downloaded the file and select the folder Use the cd command followed by the folder name to navigate into the desired folder.

cd Age_and_Gender_Prediction

In the terminal, ensure you are in the correct directory where you want to create the virtual environment. 

python -m venv venv

After creating the virtual environment, you need to activate it.

.\venv\Scripts\activate

Installing the Requirements File

Ensure that the virtual environment is active (your terminal prompt should indicate this). Install the required packages listed in the requirements.txt file using the following command:

pip install -r requirements.txt

This will install all the dependencies specified in the requirements.txt file.

Running the Python Application

After installing the required packages, you can run your Python application. Ensure the virtual environment is still active. Use the following command to run your application:

python app.py

The terminal will display the following prompt:

[Running] python -u "c:\path\to\your\project\app.py"
Running on local URL:  http://127.0.0.1:7860

To create a public link, set `share=True` in `launch()`.

Accessing the Application

Once the application is running, open a web browser. Go to the following URL to access your application

http://127.0.0.1:7860

This will open the application running on your local machine.
